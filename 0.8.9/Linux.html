2.2.1 Linux

In Debian Linux and derivatives Linux native:

Using a user with permission to run the sudo command:

In first time build: 

git clone --depth=1 https://github.com/lcgamboa/picsimlab.git 
cd picsimlab 
./picsimlab_build_all_and_install.sh

To recompile use: 

make -j4
