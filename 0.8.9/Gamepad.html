10.1.2 Gamepad

This part is a gamepad with two analog axis and 7 push buttons.

The gamepad can be controlled by keyboards keys:

  • X axis - keys ’A’ and ’D’
  • Y axis - keys ’W’ and ’S’
  • Button A - key ’I’
  • Button B - key ’L’
  • Button C - key ’K’
  • Button D - key ’J’
  • Button E - key ’E’
  • Button F - key ’O’
  • Button K - key ’R’
