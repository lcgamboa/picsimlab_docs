4.14 PICGenios

It emulates the microgenius development board PICGenios PIC18F e PIC16F Microchip that uses one PIC16F777, PIC16F877A, PIC18F452, PIC18F4520, PIC18F4550 or PIC18F4620 of PICSim backend simulator.

Board PICGenios schematics.

The code examples can be loaded in PICSimLab menu Help->Examples.

The source code of board PICGenios examples using MPLABX and XC8 compiler are in the link: board_PICGenios.
