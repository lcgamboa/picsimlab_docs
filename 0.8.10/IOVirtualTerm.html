10.5.2 IO Virtual Term

This part is a virtual serial terminal. This part can be used to read and write RX/TX pins UART signals. To use this part are don’t need to use or install one virtual serial ports on computer. Clik on terminal picture to open the terminal window. The serial communication uses the 8N1 format.
