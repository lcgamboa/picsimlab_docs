6.2.2 MPLABX Int. Debug

PICSimLab also supports using MPLABX to debug simavr. The configuration is the same as described in PICSim section: MPLABX Integrated Debug. It’s possible import and debug a Arduino sketch into MPLABX using the Arduino import plugin.
